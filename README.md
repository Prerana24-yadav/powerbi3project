# powerbi3project
The Mega Mart Store Dashboard is an analytical and interactive reporting tool developed using Power BI. It provides a comprehensive view of sales, profit, customer behavior, and product performance. The goal is to support data-driven decision-making for improving business strategies and overall store profitability.

Objectives
•	Analyze revenue and profit performance across different regions.
•	Identify top-selling categories and profitable sub-categories.
•	Understand customer purchasing behavior and payment preferences.
•	Track monthly business performance trends.
•	Provide decision-makers with real-time insights through visual analytics.
________________________________________
3. Dashboard Features
•	Revenue, Profit, Quantity, and AOV KPI Cards for quick business overview.
•	Monthly Profit Trend chart to identify seasonal patterns.
•	Sales comparison by State using bar visualization.
•	Category-wise quantity contribution using Donut Chart.
•	Customer-wise revenue analysis to identify valuable customers.
•	Payment mode breakdown for better financial understanding.
•	Sub-category profit distribution using horizontal bar charts.
•	Interactive slicers for Quarter and State selections to filter data dynamically.
